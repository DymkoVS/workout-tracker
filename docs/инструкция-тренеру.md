# Дневник тренировок — инструкция для тренера

Привет! Это приложение для ведения тренировок: вы видите своих клиентов, назначаете им тренировки, а они отмечают выполнение прямо в зале. Всё работает в браузере телефона — ничего скачивать из App Store не нужно.

---

## 1. Вход

1. Откройте в браузере телефона адрес: **dymko.ru**
2. Введите логин и пароль (они в отдельном сообщении).
3. После первого входа смените пароль: **ПРОФИЛЬ** (внизу справа) → «Сменить пароль».

Вход сохраняется — заново вводить пароль каждый день не придётся.

## 2. Установить как приложение (по желанию, 30 секунд)

Чтобы открывалось с иконки, как обычное приложение:

**iPhone (Safari):**
1. Откройте dymko.ru в Safari.
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх внизу экрана).
3. Выберите **«На экран "Домой"»** → «Добавить».

**Android (Chrome):**
1. Откройте dymko.ru в Chrome.
2. Меню (три точки справа вверху) → **«Установить приложение»** (или «Добавить на главный экран»).

На рабочем столе появится иконка — приложение открывается в полный экран, без адресной строки.

## 3. Как всё устроено

Внизу экрана — меню из пяти кнопок:

| Кнопка | Что там |
|---|---|
| **ГЛАВНАЯ** | Сводка: ваша неделя, быстрый доступ к клиентам и шаблонам |
| **ЖУРНАЛ** | Все тренировки списком |
| **+** (по центру) | Создать новую тренировку |
| **ПРОГРЕСС** | Графики и аналитика |
| **ПРОФИЛЬ** | Настройки, смена пароля, выход |

Тренерские разделы — **«Клиенты»** и «Шаблоны» — открываются с ГЛАВНОЙ (блок «Быстрый доступ») или из ПРОФИЛЯ.

## 4. Клиенты

Раздел **Клиенты** — главное рабочее место:

- Список всех ваших клиентов: кто сколько тренировался на этой неделе, у кого пропуск.
- Нажмите на клиента — откроется его карточка: статистика недели, соблюдение плана за месяц, последние тренировки.

### Добавить нового клиента

1. Клиенты → **«+ ДОБАВИТЬ КЛИЕНТА»**.
2. Введите имя, придумайте логин и временный пароль (от 6 символов).
3. Передайте клиенту логин и пароль — он заходит на dymko.ru и тренируется. Пароль он сможет сменить у себя в профиле.

### Назначить тренировку клиенту

1. Откройте карточку клиента → **«+ НАЗНАЧИТЬ ТРЕНИРОВКУ»**.
2. Заполните: название (например «Ноги»), дату, упражнения с подходами (вес × повторения).
3. Сохраните — тренировка появится у клиента в журнале как запланированная.

Клиент в зале открывает её, нажимает **«НАЧАТЬ ТРЕНИРОВКУ»** и отмечает подходы по ходу — фактический вес и повторения можно поправить прямо там, если пошло не по плану.

### Посмотреть прогресс клиента

В карточке клиента → **«ПРОГРЕСС ПО УПРАЖНЕНИЯМ»**: по каждому упражнению видно историю весов — растёт клиент или стоит на месте.

## 5. Шаблоны

Если даёте нескольким клиентам похожие программы, заведите **шаблон** (раздел «Шаблоны»):

1. **«Создать»** → название + список упражнений с подходами.
2. Откройте шаблон → **«Применить»** → выберите клиента и дату.

Готовая тренировка назначится клиенту за пару нажатий — не надо каждый раз набирать упражнения заново.

## 6. Свои тренировки

Вы тоже можете вести собственные тренировки: кнопка **«+»** внизу по центру. Журнал, прогресс и графики работают и для вас.

---

## Частые вопросы

**Клиент забыл пароль.** Пока сброс — через Владимира (администратора). Напишите ему.

**Что видит клиент?** Только свои тренировки и свой прогресс. Ваших других клиентов он не видит.

**Тренировка назначена не на тот день.** Откройте тренировку → значок карандаша вверху → поменяйте дату.

**Что-то не работает или неудобно.** Напишите Владимиру — приложение живое, дорабатывается под реальные запросы. Любой фидбэк ценен.
