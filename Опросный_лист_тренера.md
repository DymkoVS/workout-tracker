## Цель документа

Данный опросник подготовлен для настройки персонального ИИ-ассистента по тренировкам.  
Ответы тренера будут использованы для:

- автоматической подготовки карточки тренировки перед каждой сессией
- отслеживания прогресса весов и повторений по каждому упражнению
- формирования рекомендаций по следующей тренировке

*Заполнить можно кратко — ключевые цифры и логика важнее развёрнутых объяснений.*

### 2.0 Механика создания тренировки *(ключевой вопрос для реализации)*

| № | Вопрос | Ответ |
|---|--------|-------|
| **2.0** | **При создании новой тренировки (например, «Ноги») — откуда брать данные?**<br>**(а)** Копировать упражнения + веса из последней тренировки этого типа<br>**(б)** Использовать статический шаблон (фиксированный список упражнений от тренера)<br>**(в)** Комбо: шаблон определяет список упражнений, прошлая тренировка подставляет веса<br>*Это влияет на то, что именно будет построено — бот, кнопка в приложении или что-то другое.* | |

## РАЗДЕЛ 3. Прогрессия весов

| № | Вопрос | Ответ |
|---|--------|-------|
| **3.1** | **По какому принципу растёт вес от тренировки к тренировке? Есть ли конкретное правило?**<br>*Например: если сделал все повторы чисто — следующий раз +2.5 кг; или +5% раз в 2 недели* | |
| **3.2** | **Когда НЕ нужно повышать вес? Какие критерии?**<br>*Нарушение техники, меньше N повторов, субъективное состояние?* | |
| **3.3** | **При силовом цикле — какие целевые веса на жиме штанги лёжа к концу 4 недель? Есть ли план по другим упражнениям?**<br>*Сейчас рабочий вес 70 кг (1 повтор), техника ещё нечистая* | |
| **3.4** | **Как работает «смазанный» подход — нужно ли повторить тот же вес на следующей тренировке или двигаться дальше?** | |
| **3.5** | **Отказ — это желательный результат или нет? Во всех упражнениях или только в изолирующих?** | |
| **3.6** | **При плато (вес не растёт 3+ тренировки подряд) — что делать?**<br>*Снизить вес и добрать объём? Сменить упражнение? Разгрузочная неделя?* | |
| **3.7** | **Отличается ли логика прогрессии в компаундных (жим штанги, жим платформы) и изолирующих упражнениях (сведения, разгибания)?** | |
| **3.8** | **Как должна выглядеть подсказка по весу перед тренировкой?**<br>**(а)** Просто показать веса из прошлой тренировки — принять решение вручную<br>**(б)** Показать прошлый вес и автоматически предложить следующий (+2.5 кг и т.д.) по правилу прогрессии<br>*Это определяет, насколько «умным» должен быть ассистент.* | |
